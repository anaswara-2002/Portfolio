Camera-Based Intrusion Detection System (Face-Based Authorization)

Python, OpenCV (Haar Cascade), Logging

Developed a camera-based intrusion detection system that continuously scans individuals using face detection techniques. The system verifies whether the detected person is authorized; if an unauthorized individual is detected, an alert is generated. All scanning activities and intrusion attempts are recorded in log files for monitoring and analysis.
